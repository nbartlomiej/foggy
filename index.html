Usage

  1. Invoke with standard options 
     $('.your-selector').foggy();
  2. Tailor the fog according to your needs 
     $('.your-selector').foggy({ 
       blurRadius: 2, opacity: 0.5 
     });

How it works

Foggy makes copies of a selected HTML element. It adds a little transparency to all copies and moves each of them by small amount, thus creating the blur effect.

For details, open this page in a web inspector, or view the Foggy source at GitHub:
http://github.com/nbartlomiej/foggy/
